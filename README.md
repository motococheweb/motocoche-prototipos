# motocoche-prototipos

Prototipos HTML de **validación de producto** del marketplace de recambios Motocoche.
Maquetas interactivas (un solo archivo, datos de ejemplo) para revisar diseño y flujo con
el equipo antes de desarrollar. Fieles al sistema visual **ADR-diseño-001** (cabecera
carbón + naranja `#FF7F00` + Roboto + sello de garantía, WCAG 2.1 AA).

## Publicado en GitHub Pages

- **Portada:** https://motococheweb.github.io/motocoche-prototipos/
- **Búsqueda · Resultados** (Sprint 1 · US-BUS01/MC-9 + US-VEH01/MC-10): https://motococheweb.github.io/motocoche-prototipos/busqueda-resultados.html
- **Propuesta de Home** (dirección visual): https://motococheweb.github.io/motocoche-prototipos/home.html

## Contenido

| Archivo | Descripción |
|---|---|
| `busqueda-resultados.html` | Pantalla **Búsqueda · Resultados**: vehículo activo, filtros, orden, cards con ✓ Compatible / OE-IAM, y estados (resultados, cargando, vacío, referencia no compatible, sin vehículo). Barra superior "PROTOTIPO" para recorrer estados. |
| `home.html` | Propuesta de **Home** con selector de vehículo, categorías, piezas destacadas, campa y señales de confianza. |
| `index.html` | Portada con acceso a ambos prototipos. |

> Datos **mock** (vehículos y referencias de ejemplo); no conectan con catálogo real.
> **Estos prototipos son la fuente viva del diseño** (sistema visual ADR-diseño-001). El archivo de Figma *Motocoche · Marketplace UI* queda como histórico.
